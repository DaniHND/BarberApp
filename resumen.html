¿Qué es BarberApp?

BarberApp es un sistema web para administrar una barbería: citas, clientes, servicios, agenda del día, lista de espera e ingresos. Todo desde el teléfono, tablet o computadora sin instalar ninguna aplicación.

En palabras simples Imagina que eres el dueño de una barbería. Normalmente los clientes te llaman, te escriben por WhatsApp o simplemente llegan sin avisar. Con BarberApp, el cliente entra a la página web, pone su nombre, elige el servicio y la hora disponible, y listo — ya tiene su cita reservada. Tú ves en tu pantalla la agenda del día, sabes quién llega a qué hora, cuántos están esperando sin cita, cuánto llevas de ingresos y qué servicios son los más pedidos. Sin llamadas, sin papel, sin confusiones.
¿Quiénes lo usan?
  • Administrador/Barbero — gestiona la agenda y el negocio
  • Clientes — reservan citas sin crear cuenta
¿Qué problemas resuelve?
  • Elimina el control de citas por WhatsApp o papel
  • Evita que el barbero pierda tiempo con huecos vacíos
  • Permite saber quién llegó, quién no vino y quién espera
  • Lleva el registro de ingresos automáticamente
¿Cómo hace el cliente para reservar?
1. Entra a la página web 2. Escribe su nombre o apodo 3. Elige servicio, fecha y hora ¡Cita confirmada! Sin contraseña
Los 6 estados que puede tener una cita
Disponible Reservado Confirmado Atendido No Presentado En Espera
El cliente debe confirmarse llegando 10 minutos antes de su hora. Si no confirma, el espacio se libera y puede asignarse a alguien de la lista de espera.

Estructura del proyecto (planificada)
Patrón de organización del código: MVC El código sigue el patrón MVC (Modelo-Vista-Controlador). Models = la lógica que habla con la base de datos. Controllers = procesa las acciones del usuario. Views = las pantallas que ve el usuario. Es como una barbería bien organizada: el dueño (Controller) le dice al asistente (Model) que traiga los datos, y el cliente (View) recibe el resultado en pantalla.

Tecnologías usadas
PHP 8.3 — El cerebro del servidor
Procesa toda la lógica del sistema: cuando el cliente reserva una cita, PHP la valida, revisa si la hora está libre y la guarda. Usa PDO para hablar con MySQL de forma segura.
MySQL — La base de datos
Guarda citas, clientes, servicios, ingresos y configuración. 8 tablas interconectadas con claves foráneas e índices para consultas rápidas.
Tailwind CSS — El diseño
Framework CSS utilitario que genera estilos por clases. Permite construir una interfaz moderna Mobile First sin escribir CSS personalizado desde cero.
Alpine.js — Interactividad
Librería JavaScript liviana (15KB) para añadir interactividad directamente en el HTML: modales, pestañas, formularios dinámicos. Sin necesidad de Vue ni React.
Chart.js — Gráficas
Librería para mostrar gráficas visuales en el dashboard: ingresos del mes, servicios más vendidos, citas por día.
FullCalendar — Agenda visual
La agenda del barbero. Muestra las citas del día en formato de calendario interactivo con colores por estado: verde confirmado, rojo no presentado, etc.
SweetAlert2 — Notificaciones
Reemplaza los alertas del navegador con modales bonitos: confirmaciones de reserva, cancelaciones, errores de validación con diseño consistente.
